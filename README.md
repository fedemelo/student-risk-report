# Student Risk Report

[Online Report](https://noestassolo.virtual.uniandes.edu.co/estudiantes-en-riesgo)

## Local Setup

Install dependencies
```bash
yarn install
```

Run the app
```bash
yarn dev
```

And open [http://localhost:3000/estudiantes-en-riesgo](http://localhost:3000/estudiantes-en-riesgo).
